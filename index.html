ClickLok Pro

This project was to create an application which always keeps PowerPoint Applications running successfully even if external programs grab focus when someone is using a remote.

Project involved using multiple windows api’s to take input from a presentation remote and direct it to PowerPoint even if another application was in the forground. As an advanced configuration setting application can be set to run in the background on Windows boot so that the user does not need to remember to set it up.
